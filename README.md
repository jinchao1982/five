# 📓 笔记本 · 个人博客

一个简洁优雅的个人博客网站，使用纯 HTML + CSS 构建，无需任何框架或依赖。

## ✨ 特色

- 纯静态页面，加载极速
- 优雅的编辑风格排版
- 响应式布局，手机友好
- 中文衬线字体搭配西文衬线，阅读体验极佳

## 📁 文件结构

```
my-blog/
├── index.html    # 主页面
├── style.css     # 样式文件
└── README.md     # 说明文档
```

## 🚀 部署到 GitHub Pages

1. 上传所有文件到 GitHub 仓库
2. 进入仓库 **Settings → Pages**
3. Source 选择 `main` 分支，点击 **Save**
4. 几分钟后即可通过 `https://用户名.github.io/仓库名` 访问

## 🎨 自定义

修改 `style.css` 中的 CSS 变量即可快速换色：

```css
:root {
  --ink: #1a1714;       /* 主文字色 */
  --paper: #f5f0e8;     /* 背景色 */
  --accent: #b5431a;    /* 强调色 */
}
```
